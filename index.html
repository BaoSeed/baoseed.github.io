升级 Android Studio 3.1.1 出现的问题

发表于 2018-04-17 | 分类于 Android

最近因为业务需要，被公司要求做安卓，在一个老项目上改，天啊，一窍不通…

这几天都是和 Android Studio 博弈，各种错，各种搜，眼睛都快瞎了。

outputFileName

这不今天手残点了升级，又被困了半天，

搜了一下发现是 outputFileName 路径问题，升级了不支持绝对路径！！

以前代码是这样的，

去掉绝对路径，直接定义你的包名，

Execution failed for task “:app:transformClassesWithInstantRunForDebug”

Preferences->Built,Execution,Deployment->Instant Run

取消勾选

Enable Instant Run to hot swap code/resource changes on deploy(default enabled)

关于 Instant Run 可以参考这篇文章。

总结

目前没有系统的去学安卓，只能边做边学了，在这里记录一下。

每次升级还是会有很多问题的，只能慢慢摸索。
